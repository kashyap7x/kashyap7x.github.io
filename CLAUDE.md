## Overview

This is a personal academic website for Kashyap Chitta (kashyap7x.github.io). The site is generated using a Python script that processes BibTeX files containing publications and talks, then outputs a static HTML homepage. The template is based on Michael Niemeyer's lightweight homepage generator.

## Build System

The entire website is generated by a single Python script:

```bash
python build.py
```

This command:
- Reads `publication_list.bib` and `talk_list.bib`
- Processes the BibTeX entries using the `pybtex` library
- Generates `index.html` with the complete homepage

**Dependencies**: Requires `pybtex` library for BibTeX parsing.

## Architecture

### Core Components

**build.py** - Main generator script with these key functions:

- `get_personal_data()` - Returns personal information (name, email, bio text, social links, footer)
- `get_author_dict()` - Maintains mapping of author names to their personal URLs for generating hyperlinks
- `generate_person_html()` - Converts BibTeX person objects into formatted HTML with links and equal contribution markers
- `get_paper_entry()` - Transforms a publication BibTeX entry into an HTML card with image, title, authors, venue, and expandable BibTeX citation
- `get_talk_entry()` - Similar to publications but for talks/presentations
- `get_publications_html()` - Parses publication_list.bib and generates all publication entries
- `get_talks_html()` - Parses talk_list.bib and generates all talk entries
- `get_index_html()` - Assembles the complete HTML document with Bootstrap 4 styling
- `write_index_html()` - Writes the generated HTML to index.html

### Data Files

**publication_list.bib** - BibTeX file containing all publications with custom fields:
- Standard: `title`, `author`, `booktitle`, `year`
- Custom: `img` (thumbnail path), `html` (project page), `pdf`, `code`, `video`, `supp`, `poster`
- Optional: `highlight` (yellow background), `award` (red text label), `equal_contribution` (number of equal contributors)

**talk_list.bib** - BibTeX file for selected talks with fields:
- `title`, `booktitle`, `year`, `img`, `slides`, `video`

### Output Structure

**index.html** - Generated static webpage with:
- Profile section with bio and photo
- Publications section with cards (highlighted papers have yellow background)
- Selected Talks section
- Collapsible BibTeX citations for each publication
- Responsive Bootstrap 4 layout

**assets/** - Static resources:
- `assets/img/profile.jpg` - Profile photo
- `assets/img/publications/` - Publication thumbnails
- `assets/img/talks/` - Talk thumbnails
- `assets/pdf/` - PDFs, posters, talk slides
- `assets/favicon.ico` - Site favicon

## Adding New Content

**To add a publication:**
1. Add a BibTeX entry to `publication_list.bib` with all required custom fields
2. Place thumbnail image in `assets/img/publications/`
3. Add author to `get_author_dict()` in build.py if they're not already there
4. Run `python build.py` to regenerate index.html

**To add a talk:**
1. Add a BibTeX entry to `talk_list.bib`
2. Place thumbnail in `assets/img/talks/`
3. Place slides PDF in `assets/pdf/talks/` if available
4. Run `python build.py`

**To update personal information:**
- Modify `get_personal_data()` function in build.py
- Run `python build.py` to regenerate

## Highlighting and Awards

- Publications with `highlight = {true}` field get a yellow background
- Publications with `award = {text}` field display the award in red after the title
- Authors with `equal_contribution = {N}` show asterisks for the first N authors
